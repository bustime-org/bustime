Как работает bustime.ru

Предоставляемые данные являются результатом анализа GPS данных текущего местоположения транспорта. Общественный городской транспорт оснащён специальными устройствами, которые отправляют текущее местоположение техники определённое по спутникам в режиме реального времени через мобильный Интернет. Процесс состоит из следующих этапов:
  1. Данные cпутниковой навигации передаются на сервер
  2. На сервере данные записываются в базу данных и отправляются на сервер Bustime
  3. Сервер Bustime анализирует точки местонахождения транспорта и отправляет подключенным пользователям обновлённую информацию

работает на следующих устройствах:

телефонах без JavaScript/WAP
смартфонах
планшетах
ноутбуках
компьютерах
телевизорах

Введение

Сетка маршрутов и табло прибытия


Выберите нужный вам маршрут из списка. Цвет кнопки зависит от количества активного транспорта.
Если удается определить ближайшую остановку, то выводится прогнозируемое расписание прибытия транспорта для этой остановки. Вы можете выбрать другую остановку с помощью поля поиска. Направление остановки указано в названии. Остановки трамвая помечены изображением.

Обозначения

Кот появляется при клике на остановку, что устанавливает звуковое оповещение по приезду транспорта. Эти обозначения видят все пользователи сайта, и, например, водитель может увидеть что вы ждете транспорт на данной остановке.
Чашка чая обозначает транспортное средство на конечной остановке.
Луна - ночное время, данные сервера не обновляются.

Кнопка выбора маршрута

В центре кнопки выбора указано название маршрута.
Ниже - количество транспорта в одну и другую сторону.
Иконка типа транспорта означает трамвай или троллейбус.
Значок процента для маршрутов со скидками на пересадку.

Схема маршрута


На схеме маршрута отображены все остановки. Местонахождение транспорта (автобуса, трамвая, троллейбуса) отображается изображением. Рядом с названием остановки указано прогнозируемое время прибытия.

Если нажать на остановку, то вы установите звуковое оповещение по прибытию. Это также будет отображено в виде кота, ожидающего автобус, которого видят все пользователи сайта, включая водителей.

Дополнительные возможности

Откройте настройки с помощью кнопки сверху чтобы установить дополнительные параметры. Здесь вы можете:
  • отключить звук
  • отключить оповещения об изменении количества единиц транспорта на маршруте
  • отключить автопоиск текущего местонахождения по GPS (ближайшие к вам остановки автоматически появляться не будут)
  • очистить список любимых маршрутов
Для Красноярска доступен виджет баланса транспортной карты, введите номер карты сверху чтобы видеть обновляемый баланс.
